## User Register

Sistema backend creado para una prueba de desarrollador web junior.

## install:

    - git clone https://github.com/givespa/UserRegisterBackend.git
    - composer update
    - Create databse in MySQL, name: db
    - create file .env
    - copy data of .env-example and config your db
    - php artisan migrate

## Run:

    - php artisan serve
